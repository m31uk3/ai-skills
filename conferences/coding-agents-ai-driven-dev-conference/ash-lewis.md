## Background

Ash Lewis is the founder and CEO of Fastino Labs, a company building task-optimized language models for enterprise developers. Before Fastino, Ash launched DevGPT, a community for developers using AI for autonomous coding that attracted clients like Spotify, DoorDash, and IBM. That experience — and spending roughly $1 million per year on OpenAI API costs — motivated him to build a more cost-effective approach to AI model deployment.

## Key Contributions

- **Fastino's novel architecture**: Developed a new model architecture that blends elements of traditional transformers with proprietary innovations, enabling task-specific models that run on CPUs without GPUs.
- **Task-optimized LLMs**: Pioneered an approach where small, specialized models each handle one task (e.g., data redaction, document summarization) to an exceptionally high standard, trading generality for performance and efficiency.
- **DevGPT**: Built a developer community for AI-powered autonomous coding, demonstrating early traction with major tech companies.

## Fastino Labs — Key Details

- **Funding**: $7M pre-seed led by Insight Partners and M12 (Microsoft's Venture Fund), with participation from NEA, CRV, Valor, and GitHub CEO Thomas Dohmke. Later raised $17.5M led by Khosla Ventures — total funding approximately $24.5M.
- **Product**: Task-optimized language models claimed to be up to 1,000x faster than leading general-purpose models. Models are designed to run without GPUs, on standard CPUs.
- **Team**: Engineers from Stanford, Google, Microsoft, and Apple.
- **Pitch**: "Larger models can do any task to a reasonable standard, whereas our task-optimized models can only do one task but to an exceptionally high standard."
