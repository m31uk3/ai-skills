## About Semgrep

Semgrep is a leading code security solution founded in 2017 by Drew Dennison, Isaac Evans, and Luke O'Malley. The platform is used by companies like Figma, Dropbox, Slack, and Snowflake. Key metrics:
- 18,000+ organizations secured globally
- Nearly 1 million developers in user base
- 75 million scans processed in 2025
- 740,000 vulnerabilities autofixed
- AI-powered triage: 5.3 million decisions with 95% reviewer agreement
