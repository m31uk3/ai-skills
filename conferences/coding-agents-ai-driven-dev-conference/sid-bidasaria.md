## Background

Sid Bidasaria is a software engineer at Anthropic and a founding member of the Claude Code team. He joined Anthropic in November 2024, where he discovered an early prototype of what would become Claude Code — a terminal-based interface for Claude built by his colleague Boris Cherny. He immediately saw its potential and became the second engineer on the project.

Prior to Anthropic, Sid built products at Robinhood, Rubrik, and Harness, giving him broad experience across fintech, data security, and DevOps.

## Key Contributions

- **Claude Code co-creator**: Engineer #2 on the Claude Code team and creator of Claude Code subagents. Helped grow the team from 2 engineers to a full-fledged product organization with engineering, PM, design, and data science.
- **Claude Code SDK**: Demonstrated the Claude Code SDK and a new GitHub Action at Anthropic's "Code with Claude" event, showcasing tools that move AI from a simple assistant to an active agent within the software development lifecycle.
- **Product philosophy**: Advocates for stripping tools as models improve — keeping Claude Code lean enough to adapt to each new generation of capabilities.
